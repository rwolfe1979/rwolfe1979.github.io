Confluence Space Architecture

Confluence only works if people can find things and trust what they find. I designed the space structure, page templates, and permission model for a 300-user environment across 20 departments. This covers how the architecture was planned, how it was rolled out, and what it took to get teams to actually use it.

Details in this project have been generalized to protect confidential information. The system described is based on work I designed and built in a production environment.
