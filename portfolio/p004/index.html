Permission Scheme Design and User Access Management

Managing access for 300+ users across 20 departments meant the permission model had to be both tight and maintainable. I designed a scheme structure built around roles and groups rather than individuals, so access could scale without becoming a liability.

Details in this project have been generalized to protect confidential information. The system described is based on work I designed and built in a production environment.
